

## 高阶函数
- 高阶函数是对其他函数进行操作的函数，可以将他们作为参数或返回他们。
- 简单来说，高阶函数是一个函数，他接收函数作为参数或者将函数作为输出返回。

map、filter、flat都是高阶函数的使用。


## 高阶函数的意义
1、函数可以作为参数被传递
2、函数可以作为返回值输出

举例如下：
```
// 参数为函数的高阶函数
function foo(f){
    if(typeof f === 'function'){
        f();
    }
}
// 调用
foo(function(){})


// 返回值为函数的高阶函数
function foo(){
    return function(){...}
}
let f = foo();
```
